Eclipse 자주 쓰이는 유용한 단축키

Eclipse 자주 쓰이는 유용한 단축키 정리(Mac)

Posted by YoungKim on Oct 23, 2018

CMD + [ or CMD + ] : 편집창 이전/다음 화면으로 이동

CMD + Alt + 왼쪽 or 오른쪽 : 편집창 이전/다음 화면으로 이동

F2 : 마우스 오버시 생성되는 툴팁 창 노출

Ctrl + M : 전체화면 토글

CMD + Shift + R : 파일 찾기

CMD + Alt + A : 여러줄 한 꺼번에 수정

CMD + L : 입력 라인 번호로 이동
