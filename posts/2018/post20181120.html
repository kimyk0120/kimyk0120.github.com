MySQL PROCEDUR를 이용한 DUMMY 데이터 삽입 기초

Posted by Start Bootstrap on November 20, 2018

MySQL 사용 시 , 더미 데이터 생성하고자 할 때
PROCEDUR 함수를 이용하여 더미 데이터 삽입을 할 때 쓰는 반복문

DELIMITER $$  /* DELIMITER는 프로시저 앞,뒤의 위치하여 안에 있는 부분은 한번에 실행될 수 있게 하는 역할을 한다. */
DROP PROCEDURE IF EXISTS loopInsert $$   // 같은 이름의 PROCEDURE를 삭제한다.
CREATE PROCEDURE loopInsert()   // loopIndert 만들기
BEGIN
  DECLARE i INT DEFAULT 1;
  WHILE i 500 DO
    INSERT INTO tb_board (idx, TITLE)
    VALUES (i, concat('제목',i));
    SET i = i + 1;
  END WHILE;
END $$
DELIMITER

CALL loopInsert; // 실행
